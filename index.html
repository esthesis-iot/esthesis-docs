esthesis IoT platform.

esthesis IoT is an open source, cloud native suite of products to deploy and manage an Internet of Things solution. It is designed to be easy to use, scalable, and secure. The platform is built on top of Kubernetes and is designed to be cloud-agnostic. It is available as a self-hosted solution, you can install in minutes using Helm.

Dashboard

The dashboard is the main page of the esthesis CORE platform. It provides an overview of the system and the devices that are connected to it. The dashboard is customisable and can be configured to show the information that is most relevant to you. Dashboard widgets can be added, removed, and rearranged to suit your needs, and you can share dashboards between users.

Device management

The device management page allows you to view and manage the devices that are connected to the esthesis CORE platform. You can see the status of each device, view detailed information about it, examine data sent by the device, see the device location on a map, and have a virtual-terminal to interact with the device through custom commands.

Provisioning packages

Provisioning packages allow you to remotely update the software running on your devices. esthesis CORE does not mandate a specific technology for the actual update to take place, but it provides the necessary mechanism and infrastructure to manage your provisioning packages, send them to your devices, and support version-aware automatic updates.

Command and Control

The command and control functionality allows you to send commands to your devices. You can send commands to individual devices or to a group of devices via tags. Commands can be fixed, such as reboot, ping, shutdown, etc., but also custom commands. The command and control page also allows you to view the status of commands that have been sent, and see the response from the devices that received a command.

Campaigns manager

The campaign manager allows you to create and manage campaigns. A campaign is a set of actions that are executed on a group of devices. You can create campaigns to update the software on your devices, change the configuration, or perform any other action that you need to do on a group of devices. The campaign manager allows you to schedule campaigns, monitor their progress, and see the results of the campaigns that have been executed.

Dataflows

Dataflows allow you to create complex data processing pipelines that can be used to process data from your devices. There are different types of dataflows to ingest data, process data, and persist data. Because of the modular nature of dataflows, esthesis CORE can be connected to any other system you can imagine by creating a new dataflow component.

User management & Permissions

esthesis CORE provides a comprehensive user management system that allows you to create users, assign roles, and manage permissions. You can create users with different roles and assign them permissions to access different parts of the system. The user management system is designed to be flexible and scalable, so you can create as many users as you need and assign them the permissions that are appropriate for their role.

Auditing & Logging

esthesis CORE provides a comprehensive auditing and logging system that allows you to track all activity in the system. You can see who has accessed the system, what actions they have taken, and when they have taken them. If an action results in data being changed, the audit log provides a before & after snapshot of the data.

esthesis IoT Projects.

The esthesis IoT platform consists of several projects, each one focused on solving a specific problem of the IoT landscape. Projects are designed to work together and provide a complete solution for your Internet of Things needs. Below you can find a list of the projects that are included in the esthesis IoT platform.

esthesis CORE

esthesis CORE is the main project of the esthesis IoT platform. It provides the infrastructure to deploy and manage an Internet of Things solution. esthesis CORE is designed to be easy to use, scalable, and secure. It is built on top of Kubernetes and is cloud-agnostic. esthesis CORE is available as a self-hosted solution, you can install in minutes using Helm.

Status: General Availability

esthesis AGENT

esthesis AGENT is a lightweight agent that runs on your devices and connects them to the esthesis CORE platform. The agent is designed to be easy to install and configure, provides a secure connection to the platform, and a plethora of configuration options. The agent is available for a variety of operating systems and hardware platforms as a native executable, as well as a Docker container.

Status: General Availability

esthesis EDGE

esthesis EDGE provides a lightweight runtime that can run on a variety of hardware platforms, allowing you to deploy your IoT solution closer to the devices that generate the data. esthesis EDGE can fetch data from heterogeneous systems and automatically manage it as virtual devices in esthesis CORE. Using esthesis EDGE you can bring into esthesis CORE any third-party system, software or hardware based, that cannot run the esthesis AGENT.

Status: General Availability

esthesis HISTORIAN

esthesis HISTORIAN is a time-series persistent storage based on Apache IoTDB, that is designed to store and manage the data generated by your devices. The historian is designed to be scalable, reliable, and fast, and can store data from millions of devices.

Status: Planning

esthesis UNS

esthesis UNS is an implementation of the Unified Namespace concept. UNS provides a holistic view all data sources and devices in the system, abstracting the underlying complexity of the system, creating a standard way of organising and naming data. It can act as a single "source of truth" for all data in the system, allowing third-party systems to efficiently discover and interact with data anywhere in the enterprise.

Status: Planning

esthesis GATEWAY

esthesis GATEWAY acts as a proxy or bridge between one or more esthesis AGENTs and the esthesis CORE. It can be used to aggregate data from multiple devices, and forward them in batches to esthesis CORE. The esthesis GATEWAY can also be used to provide a single point of contact for all esthesis AGENTs in a network, thus simplifying networking configuration, and filter data before sending them to esthesis CORE to comply with privacy regulations.

Status: Planning
